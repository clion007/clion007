### Hi there, I'm Clion Nieh 👋

> 热爱开源，专注于网络技术、智能DNS、容器化应用与自动化安全防护。

---

### 🚀 精选项目

| 项目 | 简介 | 技术栈 |
| :--- | :--- | :--- |
| **[ADSS](https://github.com/clion007/adss)** | 全自动 DNS 智能脚本，小巧、方便、全自动的 dnsmasq 配置工具，适用于 OpenWrt 等智能路由器。 | Shell, OpenWrt |
| **[docker-jellyfin](https://github.com/clion007/docker-jellyfin)** | 基于 Alpine 构建的 Jellyfin 媒体服务器 Docker 镜像，体积更小，修复了中文字体显示及硬件解码等问题。 | Docker, Alpine, Shell |
| **[safeline-autoblocker](https://github.com/clion007/safeline-autoblocker)** | 雷池 WAF 自动封禁恶意 IP 工具，支持自定义规则、缓存加速和后台守护。 | Python, Shell, WAF |

---

### 🛠️ 技术栈与工具

- **语言**：Shell、Python、PHP、GO
- **容器化**：Docker, Docker Compose, Alpine Linux
- **网络/路由**：OpenWrt, dnsmasq, WAF (雷池)
- **自动化**：GitHub Actions, CI/CD
